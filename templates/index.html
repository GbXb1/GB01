📡 통신 빅데이터 재식별 방지 시스템

시스템 개요

본 시스템은 개인정보보호위원회의 가이드라인을 준수하여, 통신 유동인구 빅데이터의 가명처리 적정성을 평가하고 안전한 데이터 반출을 제어하는 재식별 방지 시스템입니다.

연구 및 분석 목적으로 데이터 취득 신청 시, 조회된 데이터의 표본이 3건 이하인 경우 타 정보와의 결합을 통한 개인 재식별 위험성이 존재한다고 판단하여 데이터를 원천 차단합니다.

보안 알고리즘 K-익명성 (K=3)
가명처리 레벨 Level 3 (고도)
연동 서버 상태 Secure Running

⚠️ 시스템 연결 대기 중

아직 백엔드 분석 서버가 연동되지 않았습니다. 파이썬 Flask 서버를 실행하면 실시간 가명 결합 심사 기능이 활성화됩니다.
